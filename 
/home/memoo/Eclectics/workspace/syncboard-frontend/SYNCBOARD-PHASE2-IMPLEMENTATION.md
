
# SyncBoard Phase 2 - Implementation Document

## Overview
This document outlines the implementation plan for SyncBoard Phase 2, following Trello's collaborative board functionality. The goal is to enable real-time collaboration where all board members can create, update, and move work items instantly.

## 1. Architecture Overview

### Core Principles
- **Modular Design**: Each feature has its own directory with component, HTML, and CSS
- **Signal-based State**: Use Angular signals for reactive state management
- **Real-time First**: WebSocket for instant updates, REST for persistence
- **Mock Data First**: Frontend-only with mock data until backend is ready

### Technology Stack
- **Framework**: Angular 17+ (Standalone Components)
- **State Management**: Angular Signals
- **Drag & Drop**: @angular/cdk/drag-drop
- **Real-time**: WebSocket (native + service)
- **Styling**: SCSS with CSS variables

## 2. Data Models

### Board
```typescript
interface Board {
  id: string;
  name: string;
  workspaceId: string;
  createdAt: Date;
  ownerId: string;
  members: BoardMember[];
  isStarred?: boolean;
}
```

### BoardMember
```typescript
interface BoardMember {
  userId: string;
  email: string;
  name: string;
  role: 'owner' | 'admin' | 'member' | 'observer';
  avatar?: string;
  joinedAt: Date;
}
```

### BoardList (List/Column)
```typescript
interface BoardList {
  id: string;
  name: string;
  boardId: string;
  order: number;
  cards: Card[];
}
```

### Card
```typescript
interface Card {
  id: string;
  title: string;
  description: string;
  listId: string;
  order: number;
  labels: Label[];
  dueDate?: Date;
  assignee?: string;
  createdAt: Date;
  updatedAt: Date;
}
```

### Label
```typescript
interface Label {
  id: string;
  name: string;
  color: string; // '#ff0000', 'green', etc.
}
```

### Invitation
```typescript
interface Invitation {
  id: string;
  boardId: string;
  email: string;
  role: 'admin' | 'member' | 'observer';
  invitedBy: string;
  status: 'pending' | 'accepted' | 'declined';
  expiresAt: Date;
}
```

## 4. Component Specifications

### 4.1 BoardDetailComponent
**Purpose**: Main board view with lists and cards

**Features**:
- Display board header with title and member avatars
- Show "Share" button to invite members
- Horizontal scrollable list of columns
- Drag & drop cards between lists
- Add/Edit/Delete lists
- Add cards to lists

**Template Structure**:
```html
<div class="board-detail-container">
  <!-- Board Header -->
  <header class="board-header">
    <button class="btn-back">← Back</button>
    <h1>{{ board.name }}</h1>
    <div class="members">
      <div class="avatar" *ngFor="let member of members"></div>
    </div>
    <button class="btn-share">Share</button>
  </header>

  <!-- Lists Container -->
  <div class="lists-container" cdkDropListGroup>
    <!-- List Component for each list -->
    <app-list *ngFor="let list of lists" [list]="list"></app-list>
    
    <!-- Add List Button -->
    <button class="btn-add-list">+ Add List</button>
  </div>
</div>
```

### 4.2 CardModalComponent
**Purpose**: Card detail view for editing

**Features** (MVP - Phase 2):
- Display card title (editable)
- Display card description (editable)
- Show assignee
- Show labels
- Save/Cancel buttons
- Delete card option

**Template Structure**:
```html
<div class="modal-overlay" (click)="closeModal()">
  <div class="modal-content" (click)="$event.stopPropagation()">
    <!-- Title -->
    <input type="text" [(ngModel)]="card.title" class="title-input">
    
    <!-- Description -->
    <textarea [(ngModel)]="card.description" class="description-input" 
              placeholder="Add a more detailed description..."></textarea>
    
    <!-- Labels Section -->
    <div class="labels-section">
      <span class="label" *ngFor="let label of card.labels" 
            [style.background-color]="label.color">
        {{ label.name }}
      </span>
      <button class="btn-add-label">+ Add Label</button>
    </div>
    
    <!-- Assignee Section -->
    <div class="assignee-section">
      <span>Assigned to: {{ card.assignee || 'No one' }}</span>
    </div>
    
    <!-- Actions -->
    <div class="modal-actions">
      <button class="btn-save" (click)="saveCard()">Save</button>
      <button class="btn-delete" (click)="deleteCard()">Delete</button>
      <button class="btn-cancel" (click)="closeModal()">Cancel</button>
    </div>
  </div>
</div>
```

### 4.3 InviteModalComponent
**Purpose**: Invite members to board

**Features**:

- Role dropdown (Admin, Member, Observer)

## 5. Service Specifications

### 5.1 BoardService
**Purpose**: Board CRUD operations

```typescript
@Injectable({ providedIn: 'root' })
export class BoardService {
  // Get all boards for user
  getBoards(): Promise<Board[]>
  
  // Get single board with lists and cards
  getBoard(boardId: string): Promise<Board>
  
  // Create new board
  createBoard(board: Partial<Board>): Promise<Board>
  
  // Update board details
  updateBoard(boardId: string, updates: Partial<Board>): Promise<Board>
  
  // Delete board
  deleteBoard(boardId: string): Promise<void>
  
  // Star/unstar board
  toggleStar(boardId: string): Promise<Board>
}
```

### 5.2 CardService
**Purpose**: Card CRUD operations

```typescript
@Injectable({ providedIn: 'root' })
export class CardService {
  // Get cards for a list
  getCards(listId: string): Promise<Card[]>
  
  // Get single card
  getCard(cardId: string): Promise<Card>
  
  // Create card
  createCard(card: Partial<Card>): Promise<Card>
  
  // Update card
  updateCard(cardId: string, updates: Partial<Card>): Promise<Card>
  
  // Move card to different list
  moveCard(cardId: string, targetListId: string, order: number): Promise<Card>
  
  // Delete card
  deleteCard(cardId: string): Promise<void>
  
  // Add label to card
  addLabel(cardId: string, label: Label): Promise<Card>
  
  // Remove label from card
  removeLabel(cardId: string, labelId: string): Promise<Card>
  
  // Set assignee
  setAssignee(cardId: string, userId: string): Promise<Card>
}
```

### 5.3 ListService
**Purpose**: List CRUD operations

```typescript
@Injectable({ providedIn: 'root' })
export class ListService {
  // Get lists for board
  getLists(boardId: string): Promise<BoardList[]>
  
  // Create list
  createList(list: Partial<BoardList>): Promise<BoardList>
  
  // Update list
  updateList(listId: string, updates: Partial<BoardList>): Promise<BoardList>
  
  // Delete list
  deleteList(listId: string): Promise<void>
  
  // Reorder lists
  reorderLists(boardId: string, listIds: string[]): Promise<BoardList[]>
}
```

### 5.4 InvitationService
**Purpose**: Board invitation operations

```typescript
@Injectable({ providedIn: 'root' })
export class InvitationService {
  // Send invitation
  sendInvitation(boardId: string, email: string, role: string): Promise<Invitation>
  
  // Cancel invitation
  cancelInvitation(invitationId: string): Promise<void>
  
  // Get pending invitations for board
  getPendingInvitations(boardId: string): Promise<Invitation[]>
  
  // Accept invitation (for user receiving it)
  acceptInvitation(token: string): Promise<void>
  
  // Decline invitation
  declineInvitation(token: string): Promise<void>
}
```

### 5.5 WebSocketService
**Purpose**: Real-time synchronization

```typescript
@Injectable({ providedIn: 'root' })
export class WebSocketService {
  // Connection status
  isConnected = signal(false)
  
  // Connect to WebSocket server
  connect(boardId: string): void
  
  // Disconnect
  disconnect(): void
  
  // Join board room
  joinBoard(boardId: string): void
  
  // Leave board room
  leaveBoard(boardId: string): void
  
  // Emit card moved event
  emitCardMoved(card: Card, fromListId: string, toListId: string, newIndex: number): void
  
  // Emit card updated event
  emitCardUpdated(card: Card): void
  
  // Emit card created event
  emitCardCreated(card: Card): void
  
  // Emit card deleted event
  emitCardDeleted(cardId: string, listId: string): void
  
  // Emit list created event
  emitListCreated(list: BoardList): void
  
  // Emit member joined event
  emitMemberJoined(member: BoardMember): void
  
  // Subscribe to events
  onCardMoved(callback: (data: CardMovedEvent) => void): void
  onCardUpdated(callback: (data: Card) => void): void
  onCardCreated(callback: (data: Card) => void): void
  onCardDeleted(callback: (data: CardDeletedEvent) => void): void
  onMemberJoined(callback: (data: BoardMember) => void): void
  onMemberLeft(callback: (data: { userId: string }) => void): void
}
```

## 7. API Endpoints

### Board Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/boards` | Get all boards |
| GET | `/api/boards/:id` | Get board with lists/cards |
| POST | `/api/boards` | Create board |
| PUT | `/api/boards/:id` | Update board |
| DELETE | `/api/boards/:id` | Delete board |

### List Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/boards/:boardId/lists` | Get all lists for board |
| POST | `/api/lists` | Create list |
| PUT | `/api/lists/:id` | Update list |
| DELETE | `/api/lists/:id` | Delete list |
| PUT | `/api/boards/:boardId/lists/reorder` | Reorder lists |

### Card Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/lists/:listId/cards` | Get cards for list |
| GET | `/api/cards/:id` | Get card details |
| POST | `/api/cards` | Create card |
| PUT | `/api/cards/:id` | Update card |
| DELETE | `/api/cards/:id` | Delete card |
| PUT | `/api/cards/:id/move` | Move card to list |

### Invitation Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/boards/:boardId/invite` | Send invitation |
| DELETE | `/api/invitations/:id` | Cancel invitation |
| GET | `/api/boards/:boardId/invites` | Get pending invites |
| POST | `/api/invitations/:token/accept` | Accept invitation |
| POST | `/api/invitations/:token/decline` | Decline invitation |

## 8. WebSocket Events

### Client → Server Events
| Event | Payload | Description |
|-------|---------|-------------|
| `join-board` | `{ boardId: string }` | User joins board room |
| `leave-board` | `{ boardId: string }` | User leaves board room |
| `card-moved` | `{ cardId, fromListId, toListId, newIndex }` | Card moved |
| `card-updated` | `{ card: Card }` | Card details updated |
| `card-created` | `{ card: Card }` | New card created |
| `card-deleted` | `{ cardId, listId }` | Card deleted |
| `list-created` | `{ list: BoardList }` | New list created |
| `list-deleted` | `{ listId }` | List deleted |

### Server → Client Events
| Event | Payload | Description |
|-------|---------|-------------|
| `card-moved` | `{ cardId, fromListId, toListId, newIndex, userId }` | Broadcast card move |
| `card-updated` | `{ card: Card, userId }` | Broadcast card update |
| `card-created` | `{ card: Card, userId }` | Broadcast new card |
| `card-deleted` | `{ cardId, listId, userId }` | Broadcast card deletion |
| `member-joined` | `{ member: BoardMember }` | User joined board |
| `member-left` | `{ userId: string }` | User left board |
| `presence-update` | `{ users: { userId, status }[] }` | Online users update |
